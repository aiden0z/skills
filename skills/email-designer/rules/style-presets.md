# Design Style Presets

Three design styles that control spacing, typography emphasis, and color blocking.
Choose a style based on the email's purpose. The style does NOT change components —
it changes how you use them (spacing values, font size ratios, color usage).

## Corporate (企业风格)

Best for: internal reports, weekly updates, KPI dashboards, formal announcements

| Property | Value |
|----------|-------|
| Spacing between sections | 16-20px (compact) |
| H1 size | 24px |
| H2 size | 18px |
| Body size | 13-14px |
| Header height | 50-60px |
| Color blocking | Heavy — colored section headers, stats grid backgrounds |
| Accent usage | Section title left borders, metric highlights |
| Image usage | Minimal — data tables and charts preferred |
| Tone | Professional, information-dense |

Guidelines:
- Use stats-grid and data tables prominently
- Prefer 800px width for data-heavy content
- Use nav-bar for section anchoring when email has 4+ sections
- Keep footer minimal (copyright only)
- Use the Dashboard or Single Column layout

## Editorial (编辑风格)

Best for: newsletters, content marketing, blog digests, thought leadership

| Property | Value |
|----------|-------|
| Spacing between sections | 32-40px (generous) |
| H1 size | 28-32px |
| H2 size | 22px |
| Body size | 16px |
| Header height | 60-70px |
| Color blocking | Light — white dominant, brand color for header/CTA only |
| Accent usage | Minimal — one accent color for links and CTA |
| Image usage | Heavy — hero images, article images |
| Tone | Readable, magazine-like, breathing room |

Guidelines:
- Generous line-height: 1.6 for body text (26px on 16px)
- Use the Magazine layout for article-based content
- Hero image section is essential
- Use dividers sparingly (whitespace separates sections)
- Testimonial component fits well here
- Quote blocks and callouts for pull quotes
- Footer includes social links or "share" CTA

## Minimal (极简风格)

Best for: product updates, system notifications, transactional emails, announcements

| Property | Value |
|----------|-------|
| Spacing between sections | 24px (balanced) |
| H1 size | 24-28px |
| H2 size | 18px |
| Body size | 15px |
| Header height | Minimal or none — logo only |
| Color blocking | None — pure white background throughout |
| Accent usage | CTA button only — everything else is grayscale |
| Image usage | Optional — one product image maximum |
| Tone | Clean, focused, single-purpose |

Guidelines:
- No colored section backgrounds (all white)
- No decorative dividers (use 24px spacer rows instead)
- Text hierarchy via size and weight only, not color
- One CTA button, centered, prominent
- Use the Transactional or Announcement layout
- Footer is absolute minimum (company name, legal)
- Total email height: under 800px
